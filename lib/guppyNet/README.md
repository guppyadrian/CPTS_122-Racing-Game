# guppyNet
By Adrian Nelson  
Super simple networking!  
Written in ~1000 lines of code  
Inspired by [Socket.io](https://socket.io/) with its awesome event based stuff :)

## Client Usage
```c++
#include <network/NetworkClient.hpp>
#include <iostream>

int main()
{
    NetworkManager::Start();

    NetworkClient client;
    client.connect("localhost", 3000);

    client.on("connection", [&client]()
    {
        std::cout << "connected to server!" << std::endl;
        
        client.emit("exampleEvent", "Hello Server!");
    });
    
    client.on<std::string>("message", [](const std::string& msg){
        std::cout << "Message from server: " << msg << std::endl;
    });
    
    while (true); // main loop goes here
    
    NetworkManager::Stop();
    
    return 0;
}
```

## Server Usage
```c++
#include <network/NetworkServer.hpp>
#include <iostream>

int main()
{
    NetworkManager::Start();
    
    NetworkServer server(3000);
    server.listen();
    
    server.onConnection = [](Socket socket)
    {
        socket->on<std::string>("exampleEvent", [socket](const std::string &data)
        {
            std::cout << "got an event: " << data << std::endl;
        });
        
        socket->emit("message", "Hello Client!");
    };

    server.listen();

    while (true); // main loop goes here
    
    NetworkManager::Stop();
    
    return 0;
}
```
